Example Tufte Style

by Eamonn O'Brien-Strain

To create a Tufte-style document you use standard HTML semantic elements supplemented with several HTML components, all of whose tags start with "t-".

Your file should have the following in the head element.

<link rel="stylesheet" href="tufte/tufte.min.css" /> <script src="tufte/components.js" type="module"></script>

Your body element should contain a single article element, which should start with a single h1 element, optionally followed by a t-subtitle element. Then you add a series of section elements.

Each section element should starts with a p elements that has an initial t-newthought element containing the first few words of the section. Here is an example:

<body> <article> <h1>Page Title</h1> <t-subtitle>optional subtitle</t-subtitle> <section> <p><t-newthought>Blah blah</t-newthought> blah blah blah</p> ... </section> <section> <p><t-newthought>Blah blah</t-newthought> blah blah blah</p> ... </section> ...more sections... </article> </body>

Between paragraphs you can put other specialized elements.

One example to get a citation like this one:

Tufte’s style is known for its simplicity, extensive use of sidenotes, tight integration of graphics with text, and carefully chosen typography.

Use code like this:

<t-cite href="https://edwardtufte.github.io/tufte-css/" title="GitHub tufte-css repo README" >Tufte’s style is known for its simplicity, extensive use of sidenotes, tight integration of graphics with text, and carefully chosen typography.</t-cite>
